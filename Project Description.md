# NEXIVO: Interactive DSA Visualizer

An interactive, multi-page C++ application built with the **raylib** graphics framework to visualize and demonstrate fundamental Data Structures and Algorithms (DSA). 

## Features
* **Modular Dashboard:** A central hub (`NEXIVO`) navigating to various standalone modules.
* **Algorithm Visualization:** Visual run-throughs of recursion (Fibonacci & Factorials).
* **Data Structure Implementations:** Interactive modules demonstrating operations on Linked Lists.
* **Game Development:** A custom Snake Game built using classic structure concepts.
* **Study Guide:** An integrated module dedicated to theoretical review.

## Built With
* **Language:** C++
* **Graphics Library:** raylib
* **Build System:** Makefile / VS Code Workspace
